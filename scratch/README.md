# scratch

Throwaway hand-typed practice. **Not part of the app.** This is where you write
raw JS to nail a concept by hand, then run it. Nothing here is imported by
`apps/api` or `apps/web`.

```bash
cd scratch
node closures.js        # run any file directly
```

`"type": "module"` is set, so you can use modern `import`/`export`. Delete files
freely — this folder is meant to be disposable.

RAG, from zero

Retrieval-Augmented Generation — giving a language model the ability to search your own documents before it answers. This course teaches the spine first (what RAG is, embeddings, vector databases, chunking), then hangs every strategy off it, and lands on a real architecture decision in your own travel assistant.
